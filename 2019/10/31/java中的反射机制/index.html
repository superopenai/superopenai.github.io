反射

什么时反射

java的反射库提供了一个非常丰富且精心设计的工具集,能够分析类能力的程序被称为反射,反射机制极其强大,可以用来:

  • 在运行时分析类的能力,访问java对象的属性,方法,构造方法等
  • 在运行时查看对象(例如编写一个toString供所有类使用)
  • 实现通用的数组操作代码
  • 利用Method对象

外话

java是如何在运行时识别对象和类信息的呢?

  • 传统的RTTI( RunTime Type Information，运行时类型信息 ):假定我们在编译时就知道了所有类型;
  • 反射机制:允许我们在运行时发现和使用类的信息.

得到相应Class类对象的三种方法

在程序运行期间,java系统会为所有对象维护一个类型标识,这个信息跟踪每个对象所属的类.java虚拟机可以利用运行时类型信息选择相应的方法执行.

  • 使用getClass()方法

      Person person = new Person();
        System.out.println(person.getClass().getName());

结果为 app.domain.Person
(如果类在一个包里,包名也会作为类名的一部分)




- 使用静态方法`forName()`获得类名对应的Class对象

  ```java
   String name = "app.domain.Person";
          try {
              System.out.println(Class.forName(name).getName());
          } catch (Exception e) {
              e.getStackTrace();
          }
  结果为app.domain.Person
      (如果类名保存在字符串中,并且运行时可变,就可以使用这个办法, 但是要注意,这个方法只有在className为类名或者接口名时才能执行,否则便会抛出异常.)
  • 直接T.class(如果T是任意的类型或void)
Class c1 = Person.class;

结果同上.

如何动态创建一个类实例

除了Person person = new Person();这样得到实例之外,你还可以使用person.getClass().getDeclaredConstructor().newInstance();来获得一个实例.请注意 newInstance()方法是调用默认的无参构造函数创建的实例,如果没有无参构造函数会抛出一个异常. 如果需要传参数,必须使用Constructor类中的newInstance()方法

     * <pre>{@code
     * clazz.newInstance()
     * }</pre>
  *
     * can be replaced by
  *
     * <pre>{@code
     * clazz.getDeclaredConstructor().newInstance()
     * }</pre>
  *

      原来的newInstance()的方法自JDK9后过时,

getMethod - 返回类或接口的特定方法。其中第一个参数为方法名称，后面的参数为方法参数对应 Class 的对象。

getDeclaredMethod - 返回类或接口的特定声明方法。其中第一个参数为方法名称，后面的参数为方法参数对应 Class 的对象。

getMethods - 返回类或接口的所有 public 方法，包括其父类的 public 方法。

getDeclaredMethods - 返回类或接口声明的所有方法，包括 public、protected、默认（包）访问和 private 方法，但不包括继承的方

Constructor

Class 对象提供以下方法获取对象的构造方法（Constructor）：

  • getConstructor - 返回类的特定 public 构造方法。参数为方法参数对应 Class 的对象。

  • getDeclaredConstructor - 返回类的特定构造方法。参数为方法参数对应 Class 的对象。

  • getConstructors - 返回类的所有 public 构造方法。

  • getDeclaredConstructors - 返回类的所有构造方法

    就像”上面的如何动态创建一个类实例”所说 然后你就能使用有参构造函数 创建新实例了.

       String name = "app.domain.Personss";
        // Class c1 = Class.forName(name);
        Class c1 = Person.class;
        Person person = new Person();

        Constructor[] constructors = c1.getDeclaredConstructors();
        for (Constructor constructor : constructors) {
            System.out.println(constructor);
        }
        System.out.println("-------------------------------------");
        Constructor[] constructors2 = c1.getConstructors();
        for (Constructor constructor : constructors2) {
            System.out.println(constructor);
        }
        System.out.println("-------------------------------------");

        Constructor constructor = c1.getConstructor(String.class);
        System.out.println(constructor);
        System.out.println("-------------------------------------");
        Person person2 = (Person) constructor.newInstance("程序员");
        System.out.println(person2.getName());

Field

Class 对象提供以下方法获取对象的成员（Field）：

  • getFiled - 根据名称获取公有的（public）类成员。
  • getDeclaredField - 根据名称获取已声明的类成员。但不能得到其父类的类成员。
  • getFields - 获取所有公有的（public）类成员。
  • getDeclaredFields - 获取所有已声明的类成员。
        String name = "app.domain.Personss";

        Person person = new Person();

Field field = c1.getDeclaredField("name");
